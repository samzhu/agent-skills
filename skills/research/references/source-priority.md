# 資料來源分級

> 研究時需要判斷來源可信度、或標註參考資料時參考此文件。

## 優先順序總覽

```
🥇 官方 > 🥈 大公司 > 🥉 英文文章 > 4️⃣ 社群 > 5️⃣ 中文
```

## 搜尋語言原則

```
優先順序：英文 > 中文

原因：
1. 官方文件通常是英文
2. 英文資源更新更快
3. 技術術語原文更精確
4. 大公司部落格多為英文

中文資源使用時機：
- 用戶明確要求中文
- 作為補充說明
- 需標註對應的英文原始來源
```
